# IETF / IANA / Notice

This folder contains ANCR Working Group materials related to IETF / IANA registration surfaces for “Notice”-related artefacts used by the ISO/IEC 27560 ANCR profile extension work.

## What’s here

- Draft registration text and supporting notes intended to align ANCR artefacts with IETF/IANA conventions
- Supporting working group artefacts needed for review, iteration, and eventual submission pathways (where applicable)

## How to use

1. Start with the primary draft(s) in this folder.
2. When proposing changes, reference the exact file name and the intended registration target (e.g., media type, header, registry).
3. Keep terms consistent with the ISO/IEC 27560 ANCR profile extension package and its versioned indexes.
