# ISO/IEC 27560 ANCR Profile Extension (Index)

This directory is the GitHub-facing entry point for the **ANCR Working Group** materials that package and publish the **ISO/IEC 27560 “Digital” Consent Record Information Structure Profile** extension set (ANCR-aligned).

Use this index to navigate by version.

## What this package is

This repository segment provides a versioned specification package intended to:
- support **Transparency by Default** (controller identification and notice before any identification is demanded), and
- prevent “consent records” from being misused as **surveillance record information structures**.

## How to use (quick navigation)

1. Start with the version you are implementing (typically the newest stable version).
2. Read the version’s `index.md` for:
   - scope and terminology
   - included artefacts / extensions
   - change notes vs. prior version
3. Use that version folder as the canonical reference for issues, review comments, and citations.
